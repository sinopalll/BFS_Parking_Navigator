# BFS Parking Navigator

Program ini adalah aplikasi sederhana untuk membantu mencari jalur parkir menggunakan algoritma Breadth-First Search (BFS).

## Apa yang dilakukan program ini?

- Membaca denah area parkir dari file teks.
- Mencari rute dari titik awal menuju lokasi parkir yang diinginkan.
- Menggunakan algoritma BFS untuk menelusuri jalur secara sistematis.

## Fitur utama

- Menampilkan peta parkir.
- Mencari jalur terdekat berdasarkan langkah.
- Cocok untuk proyek sederhana tentang pencarian jalur.

## Cara menjalankan

1. Pastikan Python sudah terinstall.
2. Jalankan file program utama:
   ```bash
   python BFS_Parking_Sistem.py
   ```
3. Ikuti instruksi yang muncul di terminal.

## Teknologi yang digunakan

- Python
- Algoritma BFS
- File teks sebagai representasi denah parkir
